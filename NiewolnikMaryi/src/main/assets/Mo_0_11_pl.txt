<h3>Ave Maris Stella nowa wersja</h3>
<br/>
1. Witaj, Gwiazdo morza,
Wielka Matko Boga,
Panno zawsze czysta,
Bramo niebios błoga.
<br/>
<br/>
2. Ty, coś Gabriela
Słowem przywitana,
Utwierdź nas w pokoju,
Odmień Ewy miano.
<br/>
<br/>
3. Winnych wyzwól z więzów,
Ślepym powróć blaski.
Oddal nasze nędze,
Uproś wszelkie łaski.
<br/>
<br/>
4. Okaż, żeś jest Matką,
Wzrusz modłami swymi
Tego, co Twym Synem
Zechciał być na ziemi.
<br/>
<br/>
5. O Dziewico sławna
I pokory wzorze,
Wyzwolonym z winy
Daj nam żyć w pokorze.
<br/>
<br/>
6. Daj wieść życie czyste,
Drogę ściel bezpieczną,
Widzieć daj Jezusa,
Mieć w Nim radość wieczną.
<br/>
<br/>
7. Bogu Ojcu chwała,
Chrystusowi pienie,
Obu z Duchem Świętym
Jedno uwielbienie. Amen.
